### [TIL] 전기차 데이터 기반 AI 모델 탐색

1. 정형 데이터 분석을 위한 AI 모델
    LLM을 파인튜닝하여 사용할지, 전통적인 ML모델을 사용할지 고민한 결과, 수학적 연산 기반 추론은 전통적인 ML모델이 더 우세하다는 평이 많아 해당 모델을 사용하기로 하였습니다.

    트리 기반 앙상블 모델: 전기차 데이터와 같은 일반적인 정형 데이터를 분석하고 예측할 때, 대형 언어 모델보다 자원 효율성이 높고 우수한 성능을 보여줍니다.
    - 주요 모델: XGBoost, LightGBM, CatBoost, Random Forest

    딥러닝 기반 정형 데이터 모델: 신경망 구조를 정형 데이터에 맞게 설계한 TabNet, FT-Transformer 등의 모델도 대안으로 활용할 수 있습니다.

2. 시계열 예측 특화 모델 조사

    TFT: 시간의 흐름에 따른 패턴 변화가 중요한 전기차 데이터의 시계열 예측에 특화된 딥러닝 모델입니다. 다양한 형태의 입력 변수를 처리하여 정확도 높은 미래 예측을 수행합니다.
    - 컴퓨팅 요구 사항: 딥러닝 기반이므로 트리 앙상블 모델에 비해 컴퓨팅 파워가 더 필요하지만, 데이터 샘플링과 모델 경량화를 통해 환경에 맞게 조율할 수 있습니다.
