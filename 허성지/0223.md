TIL: 빅데이터 분산 처리 및 프로젝트 기획

1. 컨설턴트님 미팅 : 빅데이터 분산 처리의 의미 파악

내용: 분산 처리의 본질은 방대한 양의 데이터를 어떻게 효율적으로 저장하고 가공할 것인가에 대한 기술적인 부분임을 알게 되었고, 이 데이터를 통해서 어떠한 서비스를 제공하는 것이 이 프로젝트의 본질적인 주제임을 알게 됨

2. 기획 : 유튜브 루키 채널 탐지 서비스

내용: 특정 관심사 도메인(K-pop, 게임 등)의 과거 데이터를 수집하여 전처리하고, 분산 저장한 후, 알고리즘의 선택을 받아 급성장하는 루키 채널을 판별하여 트렌드 대시보드로 제공하는 서비스 기획.

데이터셋: 유튜브 Data API v3를 활용하여 수집한 대규모 채널 메타데이터 및 영상 데이터(구독자, 조회수 추이, 태그, 설명란, 댓글 등).

3. 블루스카이 봇 탐지 및 거미줄 시각화
내용: 블루스카이의 실시간 데이터를 분석하여 봇 계정을 판별하고, 봇과 타겟 유저 간의 관계를 거미줄 형태로 시각화하며 실시간 적발 레이더를 제공하는 서비스 기획.

데이터셋: kaggle의 트위터 게시물 데이터 셋, 블루스카이 Firehose API
