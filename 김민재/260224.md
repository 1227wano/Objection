# 260224
## 아이디어 1: 전기차 충전소 가용성 예측 서비스
컨셉: 전기차 충전소에 갔을 때 "지금 가면 쓸 수 있을 확률"을 알려주는 서비스. 기존에 확률 기반으로 제공하는 서비스가 없다.

미해결 이슈:
- 민간 데이터 확보: 서울시·환경부 데이터는 있는데, 민간 충전사업자 데이터를 어떻게 가져오는지 방법 없음
- 과거 데이터 신뢰도: 합성데이터만 있고 실제 과거 세션 데이터를 못 구하면 예측 모델 신뢰도 문제 
- 가격 차별화: 충전소마다 가격이 다른데, 자동결제 연동은 현실성 부족 → "특가/할인" 정보 큐레이션 쪽으로 방향 전환
- API 실시간 호출 시 오차: 충전 상태 API 호출 타이밍과 실제 상태 간 시차
## 아이디어 2: 기상·전력 B2G 과부하 관제 플랫폼
컨셉: 기상 이변 시 법정동별 전력 과부하 위험도 사전 예측 → 지자체가 취약계층 보호에 선제 대응
결정사항: B2C(에너지 효율 거주지 추천)는 버림 → B2G 단일 트랙

미해결 이슈:
- 상관관계 약함: 데이터 뜯어본 결과, 기온↔전력 최대치 간 연관성을 명확히 잡기 어려웠음. "최대 전력을 썼지만 그게 100% 기온 때문인지 알 수 없다"
- 추가 변수 필요: 습도, 일사량, 풍속, 적설량, 도시열섬, 누적기온 등
- 프론트가 지도 중심: 법정동 폴리곤 색칠이 메인
